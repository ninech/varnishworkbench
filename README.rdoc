== README

=== Requirements

* Ruby

* Bundler

* Multiprocess Web server

  * This App request pages from itself. This can result in deadlocks.

* Varnish

* A Database

=== Deployment

* Install the App

  * git clone ...

  * bundle install --deployment

  * Setup Web server

  * Setup Varnish

    * Use VCL from public/ and addapt to your environment.
